### 이벤트란?

사용자가 웹 페이지에서 수행하는 특정 작업이나 발생한 사건을 의미한다.

### 이벤트 핸들러

특정 이벤트가 발생했을 때 실행되는 함수이다.
즉, 이벤트에 반응하여 특정 동작을 수행하도록 정의된 코드이다.

### 이벤트와 이벤트 핸들러의 관계

1. 이벤트가 발생하면, 브라우저는 해당 이벤트를 감지한다.
2. 개발자는 이벤트와 연결된 핸들러(함수)를 작성하여 이벤트 발생 시 실행될 작업을 지정한다.
3. 브라우저가 이벤트를 감지하면, 연결된 이벤트 핸들러가 실행된다.
